# java-discord-kat
Kat rebuilt and remastered in 4k HD concurrent, recursive, multi-threaded, cloud based, serverless, livestreamed, headless, SEO-Based, Machine learning, AI copiloted java.

## Dependencies
 - Java 17
 - Javacord
 - Maven
 - LavaPlayer

## Permissions
Per command based permission system, role based or ID based.
